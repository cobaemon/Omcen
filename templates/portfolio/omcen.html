Omcen

{% blocktrans %} 〇 なぜ、作ろうと思ったのか
　様々な成果物を公開するにあたって、成果物ごとにアカウントを作るのはユーザに冗長な操作を強いてしまうため
　その解決策として、統合管理するOmcenを作ろうと思いました

〇 概要
　Omcenの主な機能としては、アカウントの認証周り・アカウントの管理・サービスの管理を行っています
　ちなみに、Omcenの名前の由来はラテン語で「全ての中心: Omnes centrum」からきています

〇 開発環境
　OS: Windows
　言語: Python
　フレームワーク: Django
　DB: PostgreSQL

〇 ソースコード: GitHub
〇 完成品: Omcen

〇 工夫
　認証周りにはDjango-allauthを使用することで、自ら実装するよりも短期間でなおかつセキュリティホールのリスクを軽減しました
　さらに、ソーシャルアカウントでのログインにも対応しています
