Password Box

{% blocktrans %} 〇 なぜ、作ろうと思ったのか
　日々インターネットを利用していると、アカウントを作成する機会が多く
　パスワードなどを覚えておくのが大変だったので、
　その解決策として、パスワードをなどのアカウント情報を管理するPassword Boxを作ろうと思いました

〇 概要
　Password Boxの主な機能としては、アカウント情報の管理・パスワードの自動生成があります

〇 開発環境
　OS: Windows
　言語: Python
　フレームワーク: Django
　DB: PostgreSQL

〇 ソースコード: GitHub
〇 完成品: Password Box

〇 工夫
　アカウント情報の暗号化には共通鍵を使用することで暗号化・復号化時間を短くしています

〇 利用方法
　現在Password Boxは、Omcenサービスの一部となっているため
　Omcenアカウントを持っていない又はログインしていない場合は
　サイト上部の歯車マーク又はこちらから、ログインまたは新規登録をしてください

　次にサービス一覧ページにて利用したいサービスを登録してください
　サービス一覧ページはサイト上部のサービス又はこちらから、サービス一覧

　Password Box自体の利用方法は、トップページではボックスの一覧が表示されボックスの検索・新規作成・削除ができます
　ボックスを選択することでボックスの編集ができます {% endblocktrans %}
