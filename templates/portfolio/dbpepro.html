DBPePro

{% blocktrans %} 〇 なぜ、作ろうと思ったのか
　DBの権限管理を手動で行うのは冗長な作業なため、その解決手段として作ろうと思いました

〇 概要
　DBPeProの主な機能としては、対象に対して権限の付与・削除ができます
　DBPeProの名前の由来はラテン語訳で「データベース権限管理: Database permissio procuratio」からきています

〇 開発環境
　OS: Windows
　言語:
　　フロントエンド: Dart
　　バックエンド: Python
　フレームワーク:
